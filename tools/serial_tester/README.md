# Servo Serial Tester

Python tkinter GUI，用來測試 Arduino Nano + Servo 行為。
不需要安裝 PyQt6，只需要 Python 內建的 tkinter + pyserial。

---

## 啟動

```bash
pip install pyserial
python serial_tester.py
```

## STEP 指令格式

```
STEP delay_ms angle speed duration_ms [home]
```

| 參數 | 範圍 | 說明 |
|------|------|------|
| `delay_ms` | 0–65535 | 執行前等待毫秒 |
| `angle` | 0–180 | Servo 目標角度 |
| `speed` | 1–100 | 移動速度（100=最快） |
| `duration_ms` | 0–65535 | 到達目標後停留毫秒 |
| `home` | 0 或 1（**可省略，預設 1**） | 執行完是否歸位 |

### home 參數說明

| 值 | 行為 |
|----|------|
| `1`（預設）| 執行完自動回到 0°（原本行為，向下相容）|
| `0` | 停在目標角度不歸位 |

> ⚠️ `STOP` 指令無論 home 設定為何，都會強制歸位到 0°

## 單步測試區操作

1. 設定「延遲 ms」、「角度 °」、「速度」、「停留 ms」
2. 點 **[1 執行完回 0°]** 或 **[0 停在目標角度]** 選擇 home 模式
3. 角度視覺指示器會同步更新：
   - **綠色指針 + 虛線** = home=1（執行完回 0°）
   - **琥珀色指針** = home=0（停在目標角度）
4. 點 **[▶ 送出單步]** 發送，或 **[預覽指令]** 先確認指令內容

---

## 腳本測試區操作

腳本格式：`LOOP / BEGIN / STEP / END` 原始指令，`#` 開頭為註解。

### 範例 A：按 3 次自動歸位

```
LOOP 0
BEGIN 3
STEP 2000 90 60 300 1
STEP 1000 90 60 300 1
STEP 1000 90 60 300 1
END
```

### 範例 B：連續移動不歸位，最後才回 0°

```
LOOP 0
BEGIN 3
STEP 0 45 80 200 0
STEP 0 90 80 200 0
STEP 0 135 80 300 1
END
```

### 省略 home（向下相容，等同 home=1）

```
LOOP 0
BEGIN 2
STEP 0 90 60 300
STEP 500 90 60 300
END
```

## 常見問題

**Q: 找不到 COM port？**
A: 安裝 CH340 驅動：https://www.wch.cn/downloads/CH341SER_EXE.html
插拔 USB 後點 ⟳ 刷新。

**Q: 連線成功但 Servo 沒動？**
1. 先送 `PING` 確認回應 `OK PONG`
2. 送 `STATUS` 確認狀態是 `OK IDLE`
3. 縮短 delay_ms（例如改成 500）確認 Servo 是否動
4. 確認接線：D9 訊號、5V 電源、GND 共地

**Q: ERR BUSY 錯誤？**
A: 正在執行中，先送 `STOP` 再重試。

**Q: home=0 之後 Servo 一直停在那裡怎麼辦？**
A: 送 `STOP` 強制歸位，或下一個腳本最後一步設 home=1。
